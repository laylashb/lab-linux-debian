# 🟦 Projet Linux Debian – Infrastructure réseau

Par Layla Shabi
Le 05/05/2026


## Présentation

Ce projet consiste à mettre en place une infrastructure réseau complète sous Debian afin de simuler un environnement professionnel.

Le serveur Debian joue le rôle de machine centrale et fournit plusieurs services essentiels au réseau.

---

## Objectif du projet

L’objectif est de comprendre le fonctionnement d’une infrastructure Linux en conditions réelles, en configurant et en interconnectant plusieurs services réseau.

---

## Services mis en place

- DHCP : attribution automatique des adresses IP
- DNS : résolution de noms de domaine
- NTP : synchronisation de l’heure
- UFW : sécurisation du réseau
- Samba : partage de fichiers
- NFS : partage entre machines Linux

---

## Fonctionnement global

Les services fonctionnent ensemble pour créer un réseau local complet :

- les clients reçoivent automatiquement une IP
- les noms de domaine sont résolus automatiquement
- l’heure est synchronisée sur toutes les machines
- les accès réseau sont sécurisés
- les ressources sont partagées entre machines

---

## Organisation du projet

Le projet est divisé en trois parties :

1. Découverte et prise en main de Debian  
2. Présentation et compréhension des services réseau  
3. Installation et configuration complète pas à pas  

---

## Conclusion

Ce projet permet de comprendre la mise en place d’une infrastructure réseau Linux complète et les interactions entre les différents services essentiels.
